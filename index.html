Основные оси:

  • насыщенность — wght
  • пропорция — wdth
  • курсивность — ital
  • наклон — slnt
  • оптические размеры — opsz

Кастомных осей может быть сколько угодно. Они должны быть названы 4 буквами, но заглавными: SOFT, WONK, GRAD.

Было

                    @font-face {
                      font-family: 'MyFont';
                      src: url('../fonts/link.woff2');
                      font-style: normal;
                      font-weight: 500;
                    }
                    ...
                
                    @font-face {
                      font-family: 'MyFont';
                      src: url('../fonts/link-italic.woff2');
                      font-style: italic;
                      font-weight: 500;
                    }
                    ...

Было

                    @font-face {
                      font-family: 'MyFont';
                      src: url('../fonts/link-Thin.woff2');
                      font-style: normal;
                      font-weight: 100;
                    }
                    ...
                
                    @font-face {
                      font-family: 'MyFont';
                      src: url('../fonts/link-italic-Extra-bold.woff2');
                      font-style: italic;
                      font-weight: 900;
                    }
                    ...

Стало:

                @font-face {
                  font-family: 'MyFontVF';
                  src: url('../fonts/MyFontVF.woff2') format('woff2  supports variations'),
                       url('../fonts/MyFontVF.woff2') format('woff2-variations');
                  font-weight: 200 900;
                  font-stretch: 50% 200%;
                }

Фолбэк на старые шрифты

            body {
              font-family: 'MyFont', sans-serif;
            }
            @supports (font-variation-settings: normal) {
              body {
                  font-family: 'MyFontVF', sans-serif;
              }
            }
